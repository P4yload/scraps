Сменить порт
------------
Кроме GlassFish на порту 8080 может быть ещё кто-нибудь. Надо поменять порт по умолчанию для одного из них. 

Если для GlassFish 2 это можно было сделать при создании домена из среды разработки NetBeans, то в случае 3 версии GlassFish так не получится. Чтобы изменить этот порт надо зайти в администраторскую консоль localhost:4848 (по умолчанию) и во вкладке Network Config в разделе Network Listeners поменять у http-listener-1 (или требуемого) порт на нужный. Сервер после этого желательно перезапустить.

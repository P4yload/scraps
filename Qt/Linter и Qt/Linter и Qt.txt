Установка и настройка Linter в ОС Microsoft Windows для подключения из Qt
=========================================================================

Установка Linter
----------------

Загрузить дистрибутив Линтер Бастион 6.0 с сайта http://www.linter.ru/
Для возможности загрузки необходимо зарегистрироваться

При установке выбрать "демоверсия" (рисунок 1). Путь установки лучше выбрать без пробелов и русских символов. Среди компонентов установки можно убрать галочки с пунктов:
- демонстрационная база данных
- модуль доступа к Линтер из Python
- модуль доступа к Линтер из Ruby

Далее все настройки оставлять по умолчанию. В окне "Редактирование списка серверов" ничего не добавлять, просто нажать "Сохранить"

СУБД установлена

После окончания установки в трее появится значок "Linter Instance Manager" ("Администратор СУБД Линтер"). Открыть это окно и в списке серверов создать своему компьютеру новую базу (Рисунок 2)

Далее надо настроить свою базу:
- на вкладке "Имя" задать имя и псевдоним (рисунок 3)
- на вкладке "Путь" задать путь к базе, лучше без пробелов и русских символов (рисунок 4)
- на вкладке "Администратор" сменить имя и пароль (лучше не ставить пустой пароль), например на "admin" и "1" (рисунок 5)
- на вкладке "Словари" отметить пункты ODBC и Codepages, иначе не будет принимать типы varchar и nvarchar

Запустить созданную базу (пункт "Стартовать" в контекстном меню). Отметить галочкой пункт "Локальная по умолчанию". Ввести выбранные имя и пароль ("admin" и "1"). Отметить галочкой пункт "Локальная по умолчанию" (рисунок 6)

Запустить "Администратор ODBC" ("Пуск -> Программы -> СУБД ЛИНТЕР -> Администратор ODBC" или "Панель управления -> BDE Administrator"). Перейти на вкладку "Системный DSN". Выбрать Linter 6.0 и нажать "Настройка...". В окне выбрать из списка "Database server" свою базу и установить "Transaction mode" в "autocommit" (рисунок 7). Нажать "OK", на вопрос про кодовую страницу ответить "Нет"

База настроена

-----------------------------------------------------------------------------------------------

Установка Qt
------------

Загрузить дистрибутив (offline installer) с сайта http://qt.nokia.com/downloads

При установке выбрать пусть (без пробелов и русских символов) и отметить "custom" тип установки (рисунок 8)
В списке компонентов убрать всё связанное с Harmattan, Symbian и Mobility, отметить галочкой пункт Qt Designer и обязательно отметить в самом низу пункт MinGW 4.4 (рисунок 9). Остальное тоже можно выбрать по рисунку, но это не так необходимо

Qt установлен

-----------------------------------------------------------------------------------------------

Подключение к БД из Qt
----------------------

Вкратце код:

QSqlDatabase db = QSqlDatabase::addDatabase("QODBC"); // выбор драйвера и создание соединения
db.setDatabaseName("Linter 6.0");                     // указание DSN
db.setUserName("admin");
db.setPassword("1"); 

Пример приложения проверки подключения к БД лежит в архиве bdtester.zip

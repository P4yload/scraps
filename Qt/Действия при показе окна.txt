Чтобы при появлении окна происходили какие-то действия, надо переопределить метод showEvent(QShowEvent *event):
    1) объявить его в секции protected класса окна;
    2) реализовать его.
    
Например, при появлении окна обнуляются какие-то переменные и делается неактивной кнопка:

void SomeDialog::showEvent(QShowEvent *event)
{
    // обнуление переменной
    someCount = 0;
    // сделать неактивной кнопку
    ui->someButton->setEnabled(false);
}
